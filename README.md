# AI-Security-Log-Analyzer-Agent
AI Security Log Analyzer Agent is an AI-powered cybersecurity dashboard that detects anomalous network activities using Isolation Forest and maps threats to the MITRE ATT&amp;CK framework. Integrated with Google Gemini for automated incident analysis, risk assessment, and downloadable PDF security reports through an interactive Streamlit interface.

## Features
- ML-based anomaly detection
- MITRE ATT&CK mapping
- AI-generated security reports (Gemini)
- Risk scoring and severity classification
- PDF incident report generation
- Interactive Streamlit dashboard

## Tech Stack
• Python 
• Streamlit 
• Isolation Forest 
• Google Gemini API 
• Pandas 
• MITRE ATT&CK

pip install -r requirements.txt
